# Parameter

### Menambahkan parameter

1. Klik _Add new parameter_
2. Isikan data pada kolom yang tersedia.

* _Name_, nama parameter
* _Enginnering Unit_
* _Status_, status dari chimney.

Appearance
* Color
* HTML name, nama satuan dari parameter
* HTML enginnering unit
* Decimal precision
* Minimum
* Maximum

Regulary Standards
* Enabled
* Standar value

Alarm
* Limit
* Enabled
* Notif\
diaktifkan untuk menerima notif di telegram

Data Source
* Device
* Address
* Data Type

Compulation
* Enable
* Clamp
* Input minimum
* Input maximum
* Output minimum
* Output maximum
* Value

3. Klik *save* untuk menyimpan
